# Folha de Dados: FIT-101

## Identificação

`FIT-101` é o transmissor de vazão eletromagnético responsável por medir a
vazão de entrada de água bruta na planta de tratamento.

## Justificativa de seleção

O transmissor de vazão eletromagnético é adequado para líquidos condutivos,
como água industrial. Ele é comum em plantas de tratamento de água e pode se
integrar a sistemas de automação por sinais industriais.

## Especificações técnicas

| Item | Especificação |
|---|---|
| Tag | FIT-101 |
| Serviço | Medição da vazão de água bruta |
| Tipo | Medidor de vazão eletromagnético |
| Fluido | Água industrial |
| Faixa de medição | 0 a 50 m³/h |
| Sinal de saída | 4–20 mA |
| Comunicação | HART ou Modbus RTU |
| Alimentação | 24 Vcc |
| Precisão | ±0,5% da leitura |
| Diâmetro nominal | DN50 |
| Material dos eletrodos | Aço inox 316L |
| Material do corpo | Aço carbono revestido |
| Grau de proteção | IP67 |
| Temperatura de operação | 0 a 80 °C |
| Pressão máxima | 10 bar |
| Instalação | Tubulação horizontal com trecho reto antes e depois do sensor |

## Observações de aplicação

- O instrumento deve ser instalado em trecho de tubulação completamente cheio.
- A medição é usada pela lógica de controle para monitorar a alimentação da
  planta e apoiar a operação segura da bomba principal P-101.
